## 📑 Temas de Aprendizaje

### 1. ¿Qué es un algoritmo?
[cite_start]Un **algoritmo** lo podemos definir como una serie de pasos o instrucciones de manera finita, con el propósito de solucionar una problemática o tarea en específico[cite: 9, 11, 12].

### 2. Pseudocódigos
[cite_start]Es una manera de escritura de un algoritmo con una similitud muy cercana al lenguaje humano, pero manteniendo la estructura rígida de un lenguaje de programación[cite: 13, 14, 15].

### 3. Diagrama de Flujo
Es la representación gráfica de un algoritmo o proceso. [cite_start]A diferencia del pseudocódigo, utiliza símbolos geométricos conectados entre sí por flechas para mostrar visualmente la trayectoria, facilitando el análisis de los procesos[cite: 16, 17].

### 4. Prueba de Escritorio
[cite_start]Es una técnica o prueba manual que se utiliza para verificar la veracidad de la respuesta y el proceso de ejecución de un algoritmo[cite: 18, 19].

#### 📝 Ejemplo de Prueba de Escritorio
| Datos de Entrada | Proceso | Datos de Salida |
| :--- | :--- | :--- |
| Primer Número: 1 | Suma = 2 + 4 | 6 |

### 5. Programación por Bloques
[cite_start]Es una manera didáctica que permite aprender y desarrollar software usando herramientas visuales gráficas (como figuras de legos y colores) para unirlos mediante conexiones y lograr escribir código[cite: 21, 22].
